# 🧹 DecodeLabs: Phase 1 Data Cleaning & Standardization

## 📌 Project Overview
This repository contains my Week 1 deliverables for the DecodeLabs Data Analytics Internship. The primary objective of this phase was to take a raw, unstructured e-commerce dataset and perform a rigorous structural integrity audit to prepare it for deep analysis. 

Before any statistical analysis or visualization can occur, the foundation must be mathematically sound. This phase focused on identifying null values, standardizing categorical data, and correcting datatype formatting.

## 🛠️ Methodology & Execution
* **Data Type Standardization:** Converted raw string text into functional temporal data (Dates), standard currency formats, and integer quantities.
* **Structural Auditing:** Investigated categorical overlap and standardized text entries to prevent duplicate groupings during aggregation.
* **Anomaly Flagging:** Identified initial discrepancies in product categorization (e.g., extreme price variance within single categories) to be targeted during the Phase 2 Exploratory Data Analysis.
